# Airline Passenger SQL Analytics Project ✈️

This project contains advanced SQL analysis based on an airline passenger satisfaction dataset.  
It focuses on customer experience, service quality, delays, and satisfaction behavior.

---

## Key Topics Covered

- Customer satisfaction analysis
- Service quality impact
- Delay analysis
- Segmentation & grouping
- Window functions (LAG, RANK)
- UNION ALL comparisons
- Statistical scoring
- Behavioral analytics

---

## SQL Techniques Used

- CASE WHEN (grouping & segmentation)
- CTE (Common Table Expressions)
- UNION ALL (comparative analysis)
- Window Functions:
  - LAG()
  - RANK()
- Aggregations (AVG, COUNT, SUM)
- Group By analysis
- Derived metrics (service score, satisfaction rate)

---

## Business Insights

This project helps identify:

- What drives passenger satisfaction
- Which service areas impact customer experience most
- How delays affect dissatisfaction
- Which passenger segments are most sensitive
- High-value or high-risk customer groups
